# ToDo-App

Programieraufgabe gemäß Udemy-Kurs "Spring Boot Cammpus: Vom Anfänger zum Full-Stack-Experten"

Da sich in der Zwischenzeit die Versionen der verwendeten Programme schon geändert haben müssen die Aufgaben zum Teil schon auf den aktuellen Stand gebracht werden.
Dazu gibts im Kurs keine Erklärungen sondern man muss sich das Wissen zu den Änderungen selber aneignen.

#Zur App

Es ist eine mehrsprachige ToDo-Anwendung bei der sich jeder Benutzer mit einem eigenen Account anmeldet. Das Backend ist mit Java und Spring Boot realisiert. Die Daten werden in einer H2-Datenbank verwaltet und das Frontend ist mit React realisiert.
